## Usage

`$ atdownloader URL`

## Arguments and Flags

### Arguments

| Argument | Type   | Description   |
|----------|--------|---------------|
| URL      | string | twist.moe URL |

### Flags

| Flag                        | Type   | Default Value      | Description                                               |
|-----------------------------|--------|--------------------|-----------------------------------------------------------|
| -d, --destination=directory | string | $HOME/ATDownloader | Folder to download Anime into                             |
